1° passo para resolver um problema OO, identificar as classes.

Todas classes são compostas por atributos(informações) e metodos(operações/cálculos/armazenar conteúdos)
Toda classe começa com a primeira letra maiúscula
Métodos construtores precisam ter obrigatoriamente o mesmo nome que as classes

Exercício 1:

Faça um programa que leia o nome a nota de vários alunos, diga se o aluno foi 
aprovado, e no final mostre o nome do aluno com a maior nota
A nota de aprovação é 7

1ª - Identificar as classes
Classe: aluno

2ª - Identificar atributos
- Nome e nota

3ª - Identificar métodos
- Verificar se está aprovado
